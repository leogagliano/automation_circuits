# Descrição do Projeto

## Lógica
1. O sistema é ativado por um pulso único na entrada sem retenção [ACIONA_10]
2. Duas sequências estão disponíveis: (ABAB e ABBA), as quais podem ser selecionadas ao comutar a entrada com retenção [ABAB_OFF].
   - Quando [ABAB_OFF] não está acionada, a sequência ABAB vai ser executada ao pulsar [ACIONA_10].
   - Quando [ABAB_OFF] está acionada, a sequência ABBA vai ser executada ao pulsar [ACIONA_10].
  
## Características
- **Software:** RSLogix 5000 (Rockwell Automation)
- **Linguagem de programação:** Ladder
- **CLP:** Allen-Bradley CompactLogix L23E-QBFC1B
- **Diagrama Eletropneumático:** FESTO FluidSIM
- **Fotos**: Leonardo Gagliano

# Project Description

## Logic
1. The system is activated by a single pulse on the non-retentive input [ACIONA_10].
2. Two sequences are available: (ABAB and ABBA), which can be selected by switching the retentive input [ABAB_OFF].
   - When [ABAB_OFF] isn't activated, the sequence ABAB  will be executed by pulsing [ACIONA_10].
   - When [ABAB_OFF] is activated, the sequence ABBA will be executed by pulsing [ACIONA_10].
  
## Characteristics
- **Software:** RSLogix 5000 (Rockwell Automation)
- **Programming Language:** Ladder
- **PLC:** Allen-Bradley CompactLogix L23E-QBFC1B
- **Electropneumatic Diagram:** FESTO FluidSIM
- **Project Photos**: Taken by Leonardo Gagliano
